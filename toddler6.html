English - 2021/02/21 토들러6(시간이걸리다/돈이들다)

5분 걸려요

It takes 5 minutes.

5000원 들어요.

It costs 5000 won ※ costs발음할 때 ㅆ+ㅌ+ㅆ로 해야된다

시간이 많이 걸려요.

It takes a lot time

돈이 많이 들려요

It costs a lot money

시간이 많이 걸렸어요.

It took a lot time. It took really(very) long

돈이 많이 들었어요.

It cost a lot money

시간이 많이 걸릴거예요.

It’s going to take a lot of time / a long time / long

돈이 많이 들거예요.

It’s going to cost a lot of money.

시간이 많이 안걸려요

It doesn’t take a lot of time / long / very long

시간이 많이 안걸렸어요.

It didn’t take very long.

돈이 많이 안들어요.

It doesn’t cost very much.

돈이 많이 안들었어요.

It didn’t cost very much

돈이 많이 안들거예요.

It’s not going to cost very much

Does it take a lot of time / long?

Did it take a lot of time / long?

Is it going to take a lot of time / long?

Will/Would it take long?

How long does it take?

How long did it take?

How long would it take?
