TIL - rvalue vs lvalue (feat.value categories)

rvalue VS lvalue

мӮ¬мӢӨ мқҙм „л¶Җн„° лӢӨмқҢм—җ м ңлҢҖлЎң кіөл¶Җн•ҙлҙҗм•јм§Җлқјкі  мғқк°Ғн–Ҳм—ҲлҚҳ мЈјм ңмқҙлӢӨ. л§үм—°н•ҳкІҢ rvalueлҠ” мҳӨлҘёмӘҪм—җ мҳ¬мҲҳмһҲлҠ” н‘ңнҳ„мӢқ, lvalueлҠ” мҷјмӘҪм—җ мҳ¬ мҲҳ мһҲлҠ” н‘ңнҳ„мӢқмқҙлқјкі  мғқк°Ғн•ҳкі  кө¬мІҙм ҒмңјлЎңлҠ” кіөл¶Җн•ҳм§Җм•Ҡм•ҳлӢӨ. мқҙлІҲм—җ boost::moveм—җ лҢҖн•ҙ мЎ°мӮ¬н•ҳлҚҳмӨ‘ rvalue, lvalueк°Җ м–ёкёүлҗҳм–ҙ кіөл¶ҖлҘј н•ҙліҙл Өн•ңлӢӨ.

лЁјм Җ лӮҙк°Җм•ҢлҚҳ rvalue lvalueлҠ” Cм–ём–ҙ кҙҖм җм—җм„ңмқҳ м •мқҳмқҙлӢӨ. м •нҷ•нһҲлҠ” assign operator(=)мҷјмӘҪ нҳ№мқҖ мҳӨлҘёмӘҪм—җ мҳӨлҠ” н‘ңнҳ„мӢқмқҙ lvalueмқҙкі , rvalueлҠ” мҳӨлҘёмӘҪм—җ мҳӨлҠ”н‘ңнҳ„мӢқмқҙлӢӨ. нҳ„мһ¬мқҳ C++мқҖ кҪӨлӮҳ ліөмһЎн•ң Value Categories(н‘ңнҳ„мӢқ лІ”мЈј)лҘј к°Җм§Җкі  мһҲлӢӨ. C++11 мқҙм „кіј нӣ„лЎң Value Categoriesк°Җ лӢӨлҘёлҚ°, мқҙм „мқ„ лЁјм Җ м„ӨлӘ…н•ҳмһҗл©ҙ

C++11 мқҙм „ Value Categories

лӮҙк°Җ м•Ңкі  мһҲлҚҳлҢҖлЎң(?) rvalueмҷҖ lvalueл‘җк°Җм§ҖлЎң мқҙлЈЁм–ҙм ёмһҲм—ҲлӢӨ. н•ҳм§Җл§Ң Cм–ём–ҙмҷҖлҠ” лӢӨлҘҙкІҢ lvalueлҠ” м–ҙл– н•ң л©”лӘЁлҰ¬мқҳ мң„м№ҳлҘј к°ҖлҰ¬нӮӨлҠ” н‘ңнҳ„мӢқмқҙл©° м°ёмЎ°м—°мӮ°мһҗлҘј мқҙмҡ©н•ҙ мң„м№ҳлҘј м°ёмЎ°н•  мҲҳ мһҲлӢӨ. rvalueлҠ” lvalueк°Җ м•„лӢҢ н‘ңнҳ„мӢқмқҙлӢӨ.

C++11 мқҙнӣ„мқҳ Value Categories

C++11мқҙнӣ„м—җлҠ” Value Categoriesк°Җ кҪӨ ліөмһЎн•ҳкІҢ л°”лҖҢм—ҲлҠ”лҚ°, identityлқјлҠ” мӢқлі„мһҗлҘј LValueк°Җ к°Җм§Җкұ°лӮҳ к°Җм§Җм§Җм•Ҡкұ°лӮҳм—җ л”°лқј lvalueмқём§Җ rvalueмқём§Җ лӮҳлүңлӢӨ. к·ёлҰ¬кі  move semanticмқҙлқјлҠ” кІғмқҙ лҸ„мһ…лҗҳм–ҙ, moveк°Җ к°ҖлҠҘн•ң н‘ңнҳ„мӢқмқём§Җ м•„лӢҢм§Җм—җ л”°лқјм„ңлҸ„ Valueк°Җ лӮҳлүңлӢӨ. мҰү identityлҘј к°Җм§Җкі мһҲкі  moveк°Җ к°ҖлҠҘн•ң н‘ңнҳ„мӢқ identityлҘј к°Җм§Җм§Җм•Ҡкі  moveк°Җ к°ҖлҠҘн•ң н‘ңнҳ„мӢқ identityлҘј к°Җм§Җм§Җм•Ҡкі  moveк°Җ л¶Ҳк°ҖлҠҘн•ң н‘ңнҳ„мӢқ identityлҘј к°Җм§Җкі  moveк°Җ к°ҖлҠҘн•ң н‘ңнҳ„мӢқ мқҙл ҮкІҢ л„Өк°Җм§ҖлЎң ValueлҘј лӮҳлҲҢ мҲҳ мһҲлӢӨ. н•ҳм§Җл§Ң identityлҸ„ к°Җм§Җм§Җм•Ҡкі  moveк°Җ л¶Ҳк°ҖлҠҘн•ң н‘ңнҳ„мӢқмқҖ мһҲмқ„ мҲҳ м—Ҷкё°м—җ м„ёк°Җм§ҖлЎң лӮҳлҲҢ мҲҳмһҲлӢӨ. к·ёлҰ¬кі  identityлҘј к°Җм§ҖлҠ” н‘ңнҳ„мӢқ, moveк°Җ к°ҖлҠҘн•ң н‘ңнҳ„мӢқ мқҙл ҮкІҢ лӢӨмӢң лӮҳлҲ м„ң мҙқ 5к°Җм§ҖлЎң лӮҳлҲҢ мҲҳ мһҲлӢӨ. н‘ңлЎң н‘ңнҳ„н•ҳмһҗл©ҙ м•„лһҳмҷҖ к°ҷлӢӨ.

н•ҳлӮҳн•ҳлӮҳ м•Ңм•„ліҙмһҗ.

lvalue

lvalueлҠ” identityлҘј к°Җм§Җм§Җл§Ң moveк°Җ л¶Ҳк°ҖлҠҘн•ң н‘ңнҳ„мӢқмқҙлӢӨ. lvalueмқҳ нҠ№м§•мқҖ м°ёмЎ°м—°мӮ°мһҗ(&)мқҳ н”јм—°мӮ°мһҗлЎң мӮ¬мҡ©мқҙ к°ҖлҠҘн•ң кІғмқҙлӢӨ.

int a = 10;
int *b = &a;
++a;
int c[4];
a ? b : c;

мҳҲлҘј л“Өл©ҙ lvalueлҠ” мқҙлҹ¬н•ң кІғл“Өмқҙ мһҲлӢӨ.

xvalue

xvalueлҠ” identityлҘј к°Җм§Җкі  moveк°Җ к°ҖлҠҘн•ң н‘ңнҳ„мӢқмқҙлӢӨ. xvalueлҠ” eXpiring valuesм—җм„ң л”°мҷҖ л§ҢлЈҢлҗҳлҠ” н‘ңнҳ„мӢқмқ„ лң»н•ңлӢӨкі  н•ңлӢӨ.

struct A {
  int m;
};
A&& operator+(A, A);
A&& f();
A a;
A&& ar = static_cast<A&&>(a);

м—¬кё°м„ң н‘ңнҳ„мӢқ f()мҷҖ f().m, static_cast<A&&>(a) к·ёлҰ¬кі  a + aк°Җ xvaluesлқјкі  н•ңлӢӨ.

prvalue

prvalueлҠ” identityмқ„ к°Җм§Җм§Җм•Ҡкі  мқҙлҸҷмқҙ к°ҖлҠҘн•ң н‘ңнҳ„мӢқмқҙлӢӨ. pure rvalueлқјлҠ” мқҳлҜём—җм„ң prvalueмқҙлӢӨ. C++11мқҙм „мқҳ rvalue к°ҷлӢӨкі  ліҙл©ҙлҗңлӢӨ. нӣ„мң„ мҰқк°җм—°мӮ°мһҗ, л¬ёмһҗм—ҙлҰ¬н„°лҹҙмқ„ м ңмҷён•ң лӘЁл“  лҰ¬н„°лҹҙмқҙ prvalueм—җ мҶҚн•ңлӢӨ.

glvalue

glvalueлҠ” identityлҘј к°Җм§Җкі мһҲкё°л•Ңл¬ём—җ, лҚ°мқҙн„°лҘј м ҖмһҘ н•  мҲҳ мһҲлҠ” л©”лӘЁлҰ¬мқҳ мң„м№ҳм •ліҙлҘј к°Җм§„лӢӨ.

rvalue

rvalueлҠ” moveк°Җ к°ҖлҠҘн•ң лӘЁл“  н‘ңнҳ„мӢқмқ„ лӮҳнғҖлӮёлӢӨ.
