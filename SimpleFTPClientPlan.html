이전 프로젝트에서 FTP를 이용해서 결과파일을 다른 서버로 보내는 부분이 있어서, 처음으로 네트워크 관련해서 조금 건드려보게 되었다.

이번에 조사하고 알게 된 것을 토대로 간단한 프로그램을 만들어보고자 한다.

기능 요건

  1. GUI프로그래밍일것 -> C++/CLI
  2. 파일전송 기능이 있을것
  3. 파일전송 위치 변경 가능할 것
  4. 주고 받는 로그를 실시간으로 표시할 것
  5. 로그를 파일로도 남길 것
  6. ACTIVE MODE / PASSIVE MODE 변경 가능할 것
  7. 상대 파일 List와 내 파일 List를 확인 가능할 것 -> optional or Version 2.0

비기능 요건

  1. 기본설계는 sequence diagram으로 만들 것
  2. 상세설계는 class diagram과 Activity diagram으로 만들것
  3. 테스트는 운용테스트 수준으로 할것이나, 시간이 된다면 google test를 사용하여 unit test를 진행할 것

이렇게 요건을 정의해도 되는지는 아직 잘 모르겠지만, 이정도로 프로그램을 만들어 보려고 한다.
