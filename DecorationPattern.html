위키피디아 정의는 아래와 같다. 데코레이터 패턴(Decorator pattern)이란 주어진 상황 및 용도에 따라 어떤 객체에 책임을 덧붙이는 패턴으로, 기능확장이 필요할 때 서브클래싱 대신 쓸 수 있는 유연한 대안이 될 수 있다. Head First Design Pattern책에서의 정의는 다음과 같다. 데코레이터 패턴에서는 객체에 추가적인 요건을 동적으로 첨가한다. 데코레이터는 서브클래스를 만드는 것을 통해서 기능을 유현하게 확장할 수 있는 방법을 제공한다.

솔직히 정의를 봐도 전혀 모르겠다. 동적으로 객체를 추가 가능하고, 서브클래스 즉 상속을 이용해서 기능확장을 한다는 것은 알겠다.

구조를 한번 보자.

구조

  • Component
    상속받는 ConcreateComponent와 Decorator의 공통기능을 정의하고, 실제 클라이언트가 사용하는 클래스이다.
  • ConcreateComponent
    기본적인 기능, 이기능에 장식이 붙는 형식
  • Decorator
    추가적인 기능의 공통기능을 정의한다.
  • ConcreateDecorator
    추가적인 기능, 기본적인 기능에 장식으로 추가적인 기능을 붙인다.

예

class Decorator : public Product 
{
public:
	Decorator(Product* pd) :product(pd) {};
	virtual std::string description() = 0;
	Product* product;
};

class PartsA : public Decorator
{
public:
	PartsA(Product* pd): Decorator(pd){
		
	}
	~PartsA() {};
	std::string description() override
	{
		return product->description() + " + PartsA";
	}

	int getLeadTime() override
	{
		return product->getLeadTime() + 4;
	}
};

class PartsB : public Decorator
{
public:
	PartsB(Product* pd) :Decorator(pd) {

	}
	~PartsB() {};
	std::string description() override
	{
		return product->description() + " + PartsB";
	}

	int getLeadTime() override
	{
		return product->getLeadTime() + 2;
	}
};

class PartsC : public Decorator
{
public:
	PartsC(Product* pd) : Decorator(pd) {

	}
	~PartsC() {};
	std::string description() override
	{
		return product->description() + " + PartsC";
	}

	int getLeadTime() override
	{
		return product->getLeadTime() + 6;
	}
};

class ProductA : public Product
{
public:
	std::string description() override
	{
		return "ProductA = ";
	}
	int getLeadTime() {
		return 1;
	}
};
class ProductB : public Product
{
	std::string description() override
	{
		return "ProductB = ";
	}
	int getLeadTime() {
		return 3;
	}
};

int main()
{
	Product* Aproduct = new ProductA;
	Aproduct = new PartsA(Aproduct);
	Aproduct = new PartsB(Aproduct);
	std::cout << "Product : " <<  Aproduct->description().c_str() <<std::endl;
	std::cout << "LeadTiem : " <<  Aproduct->getLeadTime() << std::endl;

	Product* Bproduct = new ProductB;
	Bproduct = new PartsA(Bproduct);
	Bproduct = new PartsB(Bproduct);
	Bproduct = new PartsC(Bproduct);

	std::cout << "Product : " << Bproduct->description().c_str() << std::endl;
	std::cout << "LeadTiem : " << Bproduct->getLeadTime() << std::endl;

	return 0;
}
